﻿﻿﻿﻿﻿﻿﻿﻿﻿# 生产环境创建｜任务书集合

本目录用于把“生产环境创建”拆分为可执行、可验收、可交接的任务书，并在最后汇总形成任务交接书。

## 文档清单

- 00-主体-任务书.md
- 01-PRD-产销平衡系统.md
- 02-前端-任务书.md
- 03-后端-任务书.md
- 04-联调-任务书.md
- 05-封装发布-任务书.md
- 06-数据平台-任务书.md
- 07-AI与模型-任务书.md
- 99-任务交接书.md

## 使用方式

1. 先从「00-主体-任务书」确定范围、角色、节奏、变更与验收机制。
2. 依次完善各专业任务书中的【填写】字段，补齐交付物、接口与验收口径。
3. 联调与封装发布完成后，按「99-任务交接书」汇总证据与清单完成交接。
